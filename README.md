# Инструкция по запуску скрипта сбора ответов

## Что делает скрипт

Скрипт автоматически открывает браузер, заходит на сайт selftest.mededtech.ru под вашей учётной записью, проходит тест и сохраняет вопросы с правильными ответами в файл `answers.csv`.

---

## Шаг 1 — Установка Python

1. Откройте браузер и перейдите на сайт: **https://www.python.org/downloads/**
2. Нажмите большую жёлтую кнопку **"Download Python 3.x.x"**
3. Запустите скачанный файл
4. **Важно:** на первом экране установщика обязательно поставьте галочку **"Add Python to PATH"** (внизу окна)
5. Нажмите **"Install Now"** и дождитесь завершения

Чтобы проверить, что Python установился правильно:
- Нажмите `Win + R`, введите `cmd`, нажмите Enter
- В открывшемся чёрном окне введите: `python --version`
- Должна появиться строка вида `Python 3.12.x`

---

## Шаг 2 — Установка библиотеки Selenium

1. Откройте командную строку (`Win + R` → `cmd` → Enter)
2. Введите команду и нажмите Enter:

```
pip install selenium
```

3. Дождитесь завершения — появятся строки с надписью `Successfully installed`

> Chrome и ChromeDriver устанавливать отдельно **не нужно** — Selenium сделает это автоматически при первом запуске.

---

## Шаг 3 — Подготовка скрипта

1. Положите файлы `selenium_fmza.py`, `selenium_fmza_finished_tests.py` и `answers.csv` в удобную папку, например `C:\Users\ВашеИмя\Desktop\scripts\`
2. Откройте скрипты `selenium_fmza.py` и `selenium_fmza_finished_tests.py` любым текстовым редактором (например, Блокнотом — правой кнопкой мыши → "Открыть с помощью" → Блокнот)
3. Найдите в начале файла строки (`Ctrl + F`):

```python
username_field.send_keys("username")
...
password_field.send_keys("password")
```

4. Замените email и пароль на **свои** данные от сайта selftest.mededtech.ru
5. Сохраните файл (`Ctrl + S`)

---

## Шаг 4 — Запуск основного скрипта

1. Откройте командную строку (`Win + R` → `cmd` → Enter)
2. Перейдите в папку со скриптом командой `cd`, например:

```
cd C:\Users\ВашеИмя\Desktop\scripts
```

3. Запустите скрипт:

```
python selenium_fmza.py
```

4. Автоматически откроется окно браузера Chrome — **не закрывайте его** и **не переключайтесь между вкладками** во время работы скрипта. Иногда допустимо нажать на нужные кнопки самостоятельно, если сервер долго грузится. Например, "Перейти к первому вопросу"

5. При корректной работе сервера скрипт завершит работу самостоятельно, окно браузера закроется

---

## Шаг 5 — Открытие результатов

Файл `answers.csv` можно открыть в Excel:

1. Откройте Excel
2. Перейдите: **Данные → Из текста/CSV**
3. Выберите файл `answers.csv`
4. В параметрах импорта выберите кодировку **UTF-8** и разделитель **запятая**
5. Нажмите **Загрузить**

В файле будут столбцы:
- `question_html` — текст вопроса
- `correct_answer` — правильный ответ
- `all_answers` — все варианты ответов (правильный отмечен `[+]`)

## Что делать, если что-то пошло не так

**Скрипт упал с ошибкой сразу после запуска**
→ Убедитесь, что Chrome установлен и обновлён до последней версии

**Ошибка `ModuleNotFoundError: No module named 'selenium'`**
→ Повторите Шаг 2 — библиотека не установилась

**Ошибка `'python' is not recognized`**
→ Python не добавлен в PATH. Переустановите Python, не забыв галочку "Add Python to PATH"

**Скрипт завис и ничего не происходит больше 2 минут**
→ Закройте командную строку (`Ctrl + C`), закройте Chrome вручную и запустите заново — скрипт продолжит с того места, где остановился (уже собранные вопросы не будут собираться повторно)

**Браузер открылся, но авторизация не прошла**
→ Проверьте правильность логина и пароля в файле скрипта (Шаг 3). Иногда это проблема сервера, так что нужно просто подождать

**На сайте открылось окно с ошибкой Navigator**
→ Возможно, у вас уже есть открытый тест, который не был завершен. Закройте его вручную.

**Прохождение теста прервалось, не дойдя до завершения**
→ Какая-то ошибка на сервере. Запустите скрипт `selenium_fmza_finished_tests.py`:
```
python selenium_fmza_finished_tests.py
```
Он добавит все недособранные вопросы в итоговый файл
В командной строке будет виден прогресс, например:
   ```
   Авторизация успешна
   Найдено вариантов: 15
   Открываем вариант: ...
   Вариант обработан, переходим к следующему
   ```

---

## Важные замечания

- Во время работы скрипта **не трогайте** открытое окно Chrome
- Если нужно остановить скрипт досрочно — нажмите `Ctrl + C` в командной строке
- Повторный запуск любого из 2 скриптов **безопасен** — уже собранные вопросы не дублируются
- Для стабильной работы желательно хорошее интернет-соединение и ненагруженный компьютер
- Почему нужно открывать каждый тест вручную, когда можно было автоматизировать и это? Потому что
1. сервер работает нестабильно
2. активность на сайте выглядит менее подозриительно с ручными запусками
